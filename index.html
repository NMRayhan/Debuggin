Javascript is interpreted Language mainly it's work in google chrome V8 engine and NodeJS Runtime and others browser has own engine, Single thread procedure. everyone know interpreted means source code don't need to compiled, source code directly executed line by line top to bottom in JS file. In this concept we all know about Execution stack. Execution stack means when we any js file load any browser this time browser build a executed stack for every variable and references for every function declaration after that javascript function references call in line by line and executed this file and it's call Execution context, after that it's complete inside an environment and it's call Javascript Runtime environment (JRE), after that JRE Complete then every function call asyncronusly.
asyncronus means allow multiple works in a same time

In web browser use 2 type of storage when we use any browser first one is session-storage second one is Local-storage. in Local-storage is used for certen period and session-storage used for temporary period. session-storage destroyed when browser closed but Local-storag is saved in our browser storage, browser closed not effected this storage,
both browser store data to storage using object notaion {} and using keys value pair. both storage are same method using set and get data from storage. but when we get data from Local-storag it's JSON formate and other storage are string
some coding example of session-storage are
sessionStorage.setItem('key', 'value'); //this the basic executed code of set anythings to session-storage,
sessionStorage.getItem('key'); // this is the executed code of get anythings from session-storage
sessionStorage.removeItem('key'); // this is the executed code removeItem from session-storage
if we clear session-storage then sessionStorage.clear(); //execute this code

Local-storage almost same as session-storage
localStorage.setItem('key', 'value'); //this the basic executed code of set anythings to localStorage,
localStorage.getItem('key'); // this is the executed code of get anythings from localStorage
localStorage.removeItem('key'); // this is the executed code removeItem from localStorage
if we clear localStorage then localStorage.clear(); //execute this code
